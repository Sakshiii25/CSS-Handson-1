What is Css?

  1. CSS stands for Cascading Styling Sheets.
  2. Css describes how HTML elements are to be displayed on screen, paper or in other media.
  3. Css is not necessary Css save a lot of work. It can control the layout of multiple web page all at once
  4. External stylesheets are stored in Css file.

Css can be added to HTML documents in 3 ways:

  • Inline-by using style inside HTML elements
  • Internal-by using style elements in the head session
  • External-by using the link element to link an external CSS file.
  • The most common way to add CSS, is to keep the styles in external CSS files. However in this tutorial we will be using inline and internal styles, because this is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS class ID, the difference is that CSS class applies a style to multiple elements.ID, other hand, applies a style to one unique elements. ID is also special in that you can use a special URL to link directly to an element and it's used by Javascrip.

In CSS, selectors are used to target a specific element or range of element on a webpage.Once and element has been targeted, a style or set of style can be applied to an element.

There is a wide rage of selectors available. Two of the most commomly used are class and ID. Both are used to target elements to which the style should be applied.
